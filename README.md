# ChefBot AI

> AI-generated recipes from what's in your fridge

## Features
- Ingredient-based recipes
- Meal planning
- Grocery lists
- Dietary filters
- Cooking timer

## Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- OpenAI GPT-4o-mini
- Stripe Subscriptions
- Vercel deployment

## Quick Start

```bash
npm install
cp .env.example .env.local
# Fill in API keys
npm run dev
```
